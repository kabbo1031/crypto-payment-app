# Simple Crypto Payment App

This is a simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet
- Send crypto to another address
- View transaction status
- Learn how blockchain payments work

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- MetaMask wallet
- Internet connection
- Basic JavaScript knowledge

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Connect your wallet
5. Enter receiver address and amount
6. Send payment

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Settings  
- `wallet.js` = Wallet connect  
- `payment.js` = Payment logic  
- `ui.js` = Screen updates  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs  
- `package.json` = Project info
